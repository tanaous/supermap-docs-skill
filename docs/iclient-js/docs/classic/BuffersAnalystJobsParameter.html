Class: BuffersAnalystJobsParameter

BuffersAnalystJobsParameter

缓冲区分析任务参数类。此类用于设置缓冲区分析的数据集、分析范围、缓冲字段、缓冲距离、距离单位等参数， 还可以对分析结果的输出参数、可视化参数进行一系列设置。
