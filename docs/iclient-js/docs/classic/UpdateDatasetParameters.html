Class: UpdateDatasetParameters

UpdateDatasetParameters

数据集信息更改参数类。该类可用于更改数据集描述信息、投影坐标系等参数， 还可以更改矢量数据集的字符集、影像数据集的颜色调色板、栅格数据集的缺省像元值等参数。
