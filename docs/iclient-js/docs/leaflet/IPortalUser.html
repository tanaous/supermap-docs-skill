Class: IPortalUser

IPortalUser

SuperMap 门户中用户信息的封装类。用于管理用户资源，包括可删除，添加资源。

new IPortalUser(iportalUrl)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.iPortalUser(iportalUrl);
  
  // 弃用的写法
  new SuperMap.iPortalUser(iportalUrl);

</script>

// ES6 Import
import { IPortalUser } from '@supermapgis/iclient-leaflet';
