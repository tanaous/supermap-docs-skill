Class: GeoFeatureThemeLayer

GeoFeatureThemeLayer

地理几何专题要素型专题图层基类。此类型专题图的专题要素形状就是由 feature.geometry 决定。此类不建议直接实例化调用。

new GeoFeatureThemeLayer(name, options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.GeoFeatureThemeLayer(name, options);
  
</script>

import { GeoFeatureThemeLayer } from '@supermapgis/iclient-leaflet';
