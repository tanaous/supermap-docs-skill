Class: SingleObjectQueryJobsParameter

SingleObjectQueryJobsParameter

单对象空间查询分析任务参数类。此类用于设置叠加分析的数据集、分析模式、保留字段等参数， 还可以对分析结果的输出参数、可视化参数进行一系列设置。

new SingleObjectQueryJobsParameter(options)
