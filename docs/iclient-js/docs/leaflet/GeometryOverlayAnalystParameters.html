Class: GeometryOverlayAnalystParameters

GeometryOverlayAnalystParameters

几何对象叠加分析参数类。 该类用于设置要进行叠加分析的几何对象、叠加操作类型等参数，还可为批量叠加分析设置几何对象数组。

new GeometryOverlayAnalystParameters(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.GeometryOverlayAnalystParameters(options);
  
  // 弃用的写法
  new SuperMap.GeometryOverlayAnalystParameters(options);

</script>

import { GeometryOverlayAnalystParameters } from '@supermapgis/iclient-leaflet';
