Class: DatasetBufferAnalystParameters

DatasetBufferAnalystParameters

数据集缓冲区分析参数类。通过该类可以指定要做缓冲区分析的数据集、数据集中几何对象的过滤条件、缓冲区通用设置等参数。

new DatasetBufferAnalystParameters(options)
