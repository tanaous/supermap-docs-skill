Class: BufferDistance

BufferDistance

缓冲区分析的缓冲距离类。通过该类可以设置缓冲区分析的缓冲距离，距离可以是数值也可以是数值型的字段表达式。

new BufferDistance(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.BufferDistance(options);
  
  // 弃用的写法
  new SuperMap.BufferDistance(options);

</script>

import { BufferDistance } from '@supermapgis/iclient-leaflet';
