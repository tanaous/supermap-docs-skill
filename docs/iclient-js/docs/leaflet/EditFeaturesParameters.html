Class: EditFeaturesParameters

EditFeaturesParameters

数据集添加、修改、删除参数类。此类可设置需要创建或修改的要素集和相应的编辑类型等参数，编辑类型包括：添加、修改、删除。

new EditFeaturesParameters(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.EditFeaturesParameters(options);
  
  // 弃用的写法
  new SuperMap.EditFeaturesParameters(options);

</script>
