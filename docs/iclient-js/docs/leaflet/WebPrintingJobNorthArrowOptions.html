Class: WebPrintingJobNorthArrowOptions

WebPrintingJobNorthArrowOptions

Web 打印地图指北针参数类。此类用于设置 Web 打印地图指北针的图片元素，可传入 Base64 位图片信息，也可传入图片 URL 地址。

new WebPrintingJobNorthArrowOptions(option)
