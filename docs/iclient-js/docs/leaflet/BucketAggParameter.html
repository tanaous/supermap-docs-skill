Class: BucketAggParameter

BucketAggParameter

子聚合类查询参数设置类。此类用于设置子聚合类查询的聚合名称、聚合字段、子聚合类集合等参数。
注意：该参数仅支持数据来源为 Elasticsearch 服务的 SuperMap iServer REST 数据服务。

new BucketAggParameter(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.BucketAggParameter(options);
  
  // 弃用的写法
  new SuperMap.BucketAggParameter(options);

</script>
