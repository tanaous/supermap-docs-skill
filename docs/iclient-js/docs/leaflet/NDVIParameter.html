Class: NDVIParameter

NDVIParameter

归一化植被指数参数类。此类可用于设置归一化植被指数中的红光谱波段、近红外光谱波段、颜色表等参数。
归一化植被指数即近红外区与红光区的反射率差值/近红外区与红光区的反射率和值。可用于反应出植物冠层的背景影响。

new NDVIParameter(options)
