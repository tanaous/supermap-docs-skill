Class: DatasetSurfaceAnalystParameters

DatasetSurfaceAnalystParameters

数据集表面分析参数类。可用于设置进行表面分析的数据集和字段、查询过滤条件、返回结果、提取方法、中间结果分辨率等参数。

new DatasetSurfaceAnalystParameters(options)
