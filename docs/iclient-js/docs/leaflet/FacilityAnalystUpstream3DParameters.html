Class: FacilityAnalystUpstream3DParameters

FacilityAnalystUpstream3DParameters

上游关键设施查找资源参数类。此类用于设置上游关键设施查找中的权值字段信息、弧段或结点等参数，还可以对不确定流向是否有效进行设置。

new FacilityAnalystUpstream3DParameters(options)

指定不确定流向是否有效。指定为 true，表示不确定流向有效，遇到不确定流向时分析继续进行； 指定为 false，表示不确定流向无效，遇到不确定流向将停止在该方向上继续查找。
