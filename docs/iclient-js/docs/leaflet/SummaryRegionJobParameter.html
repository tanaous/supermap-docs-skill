Class: SummaryRegionJobParameter

SummaryRegionJobParameter

区域汇总分析任务参数类。此类用于设置区域汇总分析的数据集、分析范围、标准属性字段、 权重字段、统计方式、汇总类型、网格类型和大小等参数，还可以对分析结果的输出参数、可视化参数进行一系列设置。

new SummaryRegionJobParameter(options)
