Class: VectorClipJobsParameter

VectorClipJobsParameter

矢量裁剪分析任务参数类。此类用于设置矢量裁剪分析的数据集、裁剪分析模式、裁剪几何对象等参数， 还可以对分析结果的输出参数、可视化参数进行一系列设置。

new VectorClipJobsParameter(options)
