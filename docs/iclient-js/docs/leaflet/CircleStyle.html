Class: CircleStyle

CircleStyle

圆形要素风格类。支持设置圆形要素的半径、线颜色、线宽、线帽形状、线条交汇边角形状、填充色、填充透明度、填充形状等参数。

new CircleStyle(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.CircleStyle(options);
  
  // 弃用的写法
  L.supermap.circleStyle(options);
  
</script>

// ES6 Import
import { CircleStyle } from '@supermapgis/iclient-leaflet';
