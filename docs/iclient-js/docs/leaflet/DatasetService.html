Class: DatasetService

DatasetService

数据集信息服务类。提供方法：查询数据集集合、查询指定数据集信息，在指定数据源下新增、修改、删除数据集等。

new DatasetService(url, options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  new L.supermap.DatasetService(url, options);
  
  // 弃用的写法
  L.supermap.datasetService(url, options);
  
</script>

// ES6 Import
import { DatasetService } from '@supermapgis/iclient-leaflet';
