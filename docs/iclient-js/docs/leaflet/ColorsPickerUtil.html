Namespace: ColorsPickerUtil

ColorsPickerUtil

色带选择器工具类。用于1、创建canvas对象，2、从几种颜色中获取一定数量的渐变色。

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/leaflet/iclient-leaflet.js"></script>
<script>
  const result = L.supermap.ColorsPickerUtil.createCanvas();

  // 弃用的写法
  const result = SuperMap.ColorsPickerUtil.createCanvas();

</script>

// ES6 Import
import { ColorsPickerUtil } from '@supermapgis/iclient-leaflet';

const result = ColorsPickerUtil.createCanvas();
