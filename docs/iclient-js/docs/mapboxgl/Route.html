Class: Route

Route

路由对象类。路由对象为一系列有序的带有属性值 M 的 x，y 坐标对，其中 M 值代表该节点到路由起点的度量值， 该值可以是距离、时间或其他任何值。路由对象可以用来模拟现实世界中的公路、铁路、河流和管线等线性地物。
