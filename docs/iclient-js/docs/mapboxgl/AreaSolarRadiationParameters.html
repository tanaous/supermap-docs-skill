Class: AreaSolarRadiationParameters

AreaSolarRadiationParameters

地区太阳辐射参数类。此类用于指定太阳辐射计算的数据集（必须为栅格数据集）、地区纬度、时间模式、起始/结束日期和时间、 大气透射率、计算小时/天数的间隔等，还可以对计算结果进行一系列设置。

new AreaSolarRadiationParameters(options)
