Class: ThiessenAnalystParameters

ThiessenAnalystParameters

泰森多边形分析参数基类。该类存储了泰森多边形分析结果数据集的名称和所在的数据源、结果数据的裁剪区域、是否返回分析得到的多边形面数组等参数。

new ThiessenAnalystParameters(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/openlayers/iclient-ol.js"></script>
<script>
  new ol.supermap.ThiessenAnalystParameters(options);
  
  // 弃用的写法
  new SuperMap.ThiessenAnalystParameters(options);

</script>
