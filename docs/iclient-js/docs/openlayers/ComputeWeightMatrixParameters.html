Class: ComputeWeightMatrixParameters

ComputeWeightMatrixParameters

耗费矩阵分析参数类。根据交通网络分析参数中的耗费字段返回一个耗费矩阵。该矩阵是一个二维数组，用来存储任意两点间的资源消耗。

new ComputeWeightMatrixParameters(options)
