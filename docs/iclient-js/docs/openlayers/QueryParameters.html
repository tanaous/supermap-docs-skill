Class: QueryParameters

QueryParameters

查询参数基类。此类为地图查询提供了查询过滤条件参数、查询结果类型、查询起始记录号、资源在服务端的保存时间等参数， 还可以对返回的查询结果进行一系列设置，并且支持自定义参数。

new QueryParameters(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/openlayers/iclient-ol.js"></script>
<script>
  new ol.supermap.QueryParameters(options);
  
  // 弃用的写法
  new SuperMap.QueryParameters(options);

</script>

// ES6 Import
import { QueryParameters } from '@supermapgis/iclient-ol';
