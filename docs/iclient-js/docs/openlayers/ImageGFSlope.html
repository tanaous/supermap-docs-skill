Class: ImageGFSlope

ImageGFSlope

栅格函数对象：对 DEM 数据生成坡度渲图。坡度是地表面上某一点的切面和水平面所成的夹角，坡度值越大，表示地势越陡峭。 此类用于设置栅格函数名称、光源高度角和高程缩放系数等参数。

new ImageGFSlope(options)
