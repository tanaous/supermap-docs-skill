Class: MappingParameters

MappingParameters

分析后结果可视化的参数类。此类用于设置分布式分析结果可视化得到的专题图的子项数组、精度、分段模式、分段个数和颜色渐变模式等参数。

new MappingParameters(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/openlayers/iclient-ol.js"></script>
<script>
  new ol.supermap.MappingParameters(options);
  
  // 弃用的写法
  new SuperMap.MappingParameters(options);

</script>

// ES6 Import
import { MappingParameters } from '@supermapgis/iclient-ol';

new MappingParameters(options);
