Class: MetricsAggParameter

MetricsAggParameter

指标聚合查询参数类，此类用于设置聚合查询的聚合类型等参数，支持的聚合类型包括：平均值、最大值、最小值、求和，默认类型为平均值。
注意：该参数仅支持数据来源为 Elasticsearch 服务的 SuperMap iServer 的 REST 数据服务。

new MetricsAggParameter(options)
