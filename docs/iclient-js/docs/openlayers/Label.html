Class: Label

Label

标签专题图图层源。标签专题图是用文本形式在图层上直接显示属性表中的数据，实质上是对图层的标注。 不仅帮助用户更好地区分地物要素，同时也显示了要素的某些重要属性，如行政区划、河流、机关、旅游景点的名称、等高线的高程等。 在 SuperMap 中，通过制作标签专题图，用户可以轻松实现地图标注。
