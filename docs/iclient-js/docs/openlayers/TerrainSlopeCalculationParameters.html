Class: TerrainSlopeCalculationParameters

TerrainSlopeCalculationParameters

地形坡度计算参数类。该类用于设置地形坡向计算的数据集、地形分析基本环境设置、高程缩放系数等参数，还可对结果数据集进行一系列设置。

new TerrainSlopeCalculationParameters(options)
