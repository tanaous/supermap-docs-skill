Class: GeometryMinDistanceAnalystParameters

GeometryMinDistanceAnalystParameters

几何对象最近距离分析参数类。该类用于设置要进行最近距离分析的数据集、属性过滤条件、查询范围的最大/最小距离等参数。

new GeometryMinDistanceAnalystParameters(options)
