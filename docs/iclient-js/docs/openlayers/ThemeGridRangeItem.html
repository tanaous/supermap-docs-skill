Class: ThemeGridRangeItem

ThemeGridRangeItem

栅格分段专题图子项类。在栅格分段专题图中，将栅格值按照某种分段模式被分成多个范围段。 本类用来设置每个范围段的分段起始值、终止值、名称和颜色等。每个分段所表示的范围为 [Start,End)。
