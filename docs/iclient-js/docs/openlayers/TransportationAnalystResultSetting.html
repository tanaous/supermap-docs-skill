Class: TransportationAnalystResultSetting

TransportationAnalystResultSetting

交通网络分析结果参数类。通过该类设置交通网络分析返回的结果，包括是否返回图片、是否返回弧段空间信息、是否返回结点空间信息等。

new TransportationAnalystResultSetting(options)

Usage
// 浏览器
