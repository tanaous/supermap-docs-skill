Class: GraticuleLayer

GraticuleLayer

经纬网类。经纬网是由间隔均匀的经度线和纬度线组成的网络，用于在地图上识别各个位置的地理坐标。

new GraticuleLayer(options)

// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/maplibregl/iclient-maplibregl.js"></script>

  new maplibregl.supermap.GraticuleLayer(options);
