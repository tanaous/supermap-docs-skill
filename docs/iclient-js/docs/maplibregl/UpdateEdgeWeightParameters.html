Class: UpdateEdgeWeightParameters

UpdateEdgeWeightParameters

边的耗费权重更新服务参数类。边的耗费权重又称弧段权值，此类用于设置需要更新权值的边、 起始转向点、终止转向点以及更新后的耗费字段、耗费权重等参数。

new UpdateEdgeWeightParameters(options)
