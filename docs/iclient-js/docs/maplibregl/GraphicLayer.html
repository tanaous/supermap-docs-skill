Class: GraphicLayer

GraphicLayer

高效率点图层类。能够支持前端百万点数据的渲染，并支持 GeoJSON，TopoJSON，二维表等多种数据格式，支持修改样式， 支持属性筛选、鼠标事件等交互操作。

new GraphicLayer(id, options)
