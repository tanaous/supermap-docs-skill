Class: BufferAnalystParameters

BufferAnalystParameters

缓冲区分析参数基类。该类存储了缓冲区分析中的通用参数，包括左缓冲距离、右缓冲距离、端点类型、圆头缓冲圆弧处线段的个数信息等。

new BufferAnalystParameters(options)
