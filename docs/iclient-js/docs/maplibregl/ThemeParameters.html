Class: ThemeParameters

ThemeParameters

专题图参数类。 该类存储了制作专题所需的参数，包括数据源、数据集名称和专题图对象。

new ThemeParameters(options)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/maplibregl/iclient-maplibregl.js"></script>
<script>
  new maplibregl.supermap.ThemeParameters(options);
  
  // 弃用的写法
  new SuperMap.ThemeParameters(options);

</script>

// ES6 Import
import { ThemeParameters } from '@supermapgis/iclient-maplibregl';

new ThemeParameters(options);

fieldValuesDisplayFilterObject

Properties:
Name Type Description
values Array.<number>

待过滤的值。
fieldName string

待过滤的字段名称，只支持数字类型的字段。
fieldValuesDisplayMode string

目前为 DISPLAY/DISABLE。当为 DISPLAY 时，表示只显示以上设置的相应属性值的要素，否则表示不显示以上设置的相应属性值的要素。

joinItemsArray.<JoinItem>

设置与外部表的连接信息 JoinItem 数组。 使用此属性可以制作与外部表连接的专题图。

themesArray.<CommonTheme>

专题图对象列表。 该参数为实例化的各类专题图对象的集合。
