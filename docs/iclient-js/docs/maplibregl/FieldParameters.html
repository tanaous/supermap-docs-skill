Class: FieldParameters

FieldParameters

字段信息查询参数类。此类用于设置进行字段信息查询的数据集和数据源等参数。 可以查询获取的字段信息包括：字段名称，字段类型，字段别名，字段默认值，字段是否允许为空，字段最大长度，是否允许长度为零等。

new FieldParameters(options)
