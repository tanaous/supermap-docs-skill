Class: FacilityAnalystTraceup3DParameters

FacilityAnalystTraceup3DParameters

上游追踪资源参数类。此类用于设置上游追踪中的权值字段信息、弧段或结点等参数，还可以对不确定流向是否有效进行设置。

new FacilityAnalystTraceup3DParameters(options)
