Class: UpdateTurnNodeWeightParameters

UpdateTurnNodeWeightParameters

更新转向耗费权重服务参数类。转向耗费权重又称转向结点权值，此类用于设置需要更新的转向结点、 起始边、终止边以及更新后的耗费字段、耗费权重等参数。

new UpdateTurnNodeWeightParameters(options)
