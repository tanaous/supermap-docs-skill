Class: KnowledgeGraph

KnowledgeGraph

KnowledgeGraph 知识图谱。此类提供了创建知识图谱实例的配置项， 包括知识图谱的尺寸、中心点、缩放比例、布局、节点配置、边配置、高亮样式、拖拽设置等参数。

new KnowledgeGraph(config)

Usage
// 浏览器
<script type="text/javascript" src="https://iclient.supermap.io/dist/maplibregl/iclient-maplibregl.js"></script>
<script>
  new maplibregl.supermap.KnowledgeGraph(config);
  
</script>

// ES6 Import
import { KnowledgeGraph } from '@supermapgis/iclient-maplibregl';
