WebSymbol

Web 符号库。SuperMap iClient for MapboxGL 通过扩展 MapboxGL 的 API，实现了对 Web 符号库的支持，提供了前后端一致的 Web 符号化制图能力和自定义 Web 符号的能力。

新增 API

maplibregl.Map.prototype.loadSymbol

通过Web符号ID加载Web符号， 用于Map#addSymbol。

参数名称 类型 描述
id string、string[] Web符号ID
callback function 在符号加载完成后调用，返回符号信息；如果有错误，则返回错误参数。

Version:

11.1.1

Example

// 加载单个Web符号
map.loadSymbol('point-1', (error, symbol) => {
      if (error) throw error;
      // Add the loaded symbol with the ID 'point-1'.
      map.addSymbol('point-1', symbol);
});

// 加载多个Web符号
var symbolIds = ['point-1', 'point-2'];
map.loadSymbol(symbolIds, (error, symbols) => {
     symbols.forEach((symbol, index) => {
         symbol && map.addSymbol(symbolIds[index], symbol);
     })
});

maplibregl.Map.prototype.addSymbol

添加一个符号。该符号可以显示在地图上。Maplibre layers 的 symbol属性可以使用该符号ID。

参数名称 类型 描述
id string 符号ID
symbol object 由Maplibre Layers中的paint、layout（visibility 属性除外）组成的符号对象
参数名称 类型 描述
paint object 可选，Maplibre Layers paint
layout object 可选，Maplibre Layers layout（visibility 属性除外）

Version:

11.1.1

Example

map.addSymbol('point-1', symbol);

扩展 Maplibre Layers

在Maplibre Layers 属性的基础上新增了symbol 属性， 指定符号ID 或者 符号表达式。

符号ID

    id: "symbol",
    source: "sourceId",
    type: "symbol",
    symbol: 'point-1'
});

    version: 8,
    sources: {},
    layers: [{
        id: "symbol",
        source: "sourceId",
        type: "symbol",
        symbol: 'point-1'
