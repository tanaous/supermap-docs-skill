Class: WebPrintingJobExportOptions

WebPrintingJobExportOptions

Web 打印的输出参数类。此类用于设置 Web 打印输出的格式、分辨率、地图比例尺、地图角度和地图中心点等参数。

new WebPrintingJobExportOptions(option)
