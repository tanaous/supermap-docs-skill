Class: Graphic

Graphic

高效率点图层点要素类。高效率点图层中的点要素，支持获取或修改点要素的属性、样式、几何信息。

new Graphic(lngLat, style, attributes)

Usage

<script type="text/javascript" src="https://iclient.supermap.io/dist/maplibregl/iclient-maplibregl.js"></script>
