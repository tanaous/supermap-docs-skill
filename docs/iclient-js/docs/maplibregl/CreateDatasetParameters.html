Class: CreateDatasetParameters

CreateDatasetParameters

数据集创建参数类。此类用于设置创建的数据集的名称、类型以及数据集所在的数据源等参数， 可创建的数据集类型包括：点、线、面、文本、复合（CAD）和纯属性数据集。

new CreateDatasetParameters(options)
