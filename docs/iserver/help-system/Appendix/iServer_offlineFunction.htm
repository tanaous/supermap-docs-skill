版本切换

更新日期：2025.06

以下对 iServer 各版本中下线的功能进行说明，若您需要使用相关功能，请参考相应的替代方案。

序号 分类 功能描述 下线原因 替代方案 隶属方案 下线策略 下线时间
1 地图服务 发布工作空间中瓦片数据源配置的地图 不推荐用法。性能极低、资源占用极高 瓦片数据直接在 iServer 进行发布即可，如果需要将多个瓦片地图放到一个服务下，可通过提供者集合来实现 iServer 全版本 12.0.0.0版本开始移除 2025.06
2 地图服务 发布工作空间中 Web 数据源配置的地图 不推荐用法。性能极低、资源占用极高 Web 数据源直接在 iServer 进行发布即可，如果需要将多个 Web 数据源地图放到一个服务下，可通过提供者集合来实现 iServer 全版本 12.0.0.0版本开始移除 2025.06
3 HBase 数据源接入 将 HBase 数据库存储的数据作为数据源发布为数据或地图服务，以及注册 HBase 到数据目录进行分布式分析 过时技术栈，不再维护 使用 PostGIS、HDFS 等数据源替代 iServer标准版及以上 11.2.0版本已移除 2024.07
4 分布式空间文件引擎接入 将本地文件系统或HDFS分布式文件系统存储的基于DSF数据库引擎生成的数据发布为地图和数据服务 过时技术栈，不再维护 无 iServer标准版及以上 11.2.0版本已移除 2024.07
5 服务表述 Tianditu.com 地图表述 天地图增加了 token限制，不能再直接叠加显示 使用 iClient 等其他表述 iServer 全版本 12.0.0.0版本开始移除 2025.06
6 服务表述 矢量瓦片的 Classic  with Vector Tile 表述（基于tileFeature.json） CartoCSS矢量瓦片表述已不再维护 使用 Mapbox(mvt) 的方式表述 iServer 全版本 12.0.0.0版本开始移除 2025.06
7 矢量瓦片表述 tileFeature 资源的RJSON、JSON、XML、JSONP、FASTJSON 表述 tileFeature.json等为过时格式 推荐使用 mvt 格式（tileFeature.mvt）进行表述 iServer 全版本 12.0.0.0版本开始移除 2025.06
8 区块链服务 提供区块链空间数据的动态出图、空间查询、多用户编辑和历史追溯等功能；区块链空间数据发布与编辑 不再维护 无 iServer高级版+空间区块链服务扩展模块 12.0.0.0版本开始移除 2025.06
9 受控集群 子节点作为受控模式加入集群，受到父节点的完全控制，其自身原有的服务实例和服务管理器都将被禁用，父节点的服务实例和数据将同步到子节点 不再维护 当前支持将服务配置等信息存储到 Oracle、PostgreSQL、Kingbase、GaussDB、Vastbase、HighgoDB、GBase9c 等数据库。此外，GIS 数据需要存储到数据库或共享目录中。满足以上条件后，可以利用多节点自动同步功能实现各个节点服务和数据的同步，然后使用通用 SLB（如nginx）做负载均衡 iServer标准版及以上 12.0.0.0版本开始移除 2025.06
10 Handler(KVP)服务接口 SuperMap iServer 提供的用于与 SuperMap iServer 2008 对接的服务。 不再维护向前兼容 无 iServer 全版本 11.2.0版本已移除 2024.07
11 分布式切图服务/地图服务 不再支持的瓦片格式：FastDFS、UTFGrid 过时格式 使用 MongoDB、MBTiles、SMTiles、SVTiles 等格式代替 iServer标准版及以上 12.0.0.0版本开始移除 2025.06
