SuperMap GIS 许可用于验证 SuperMap GIS 产品的可用性。针对 Windows 平台，提供 SuperMap 许可中心（SuperMap License Center）对许可进行配置和管理；在 Linux 平台，提供命令行方式配置许可；在 AIX 平台，继续沿用 License Manager 6R 配置许可。此外，SuperMap GIS 从 7C 系列产品开始提供云许可，从 10i 版本开始提供 Web 版许可。

该章将为您介绍许可的分类、使用、应用，以及如何在 Windows、Linux、Unix 系统上配置许可，详见：

  • 许可介绍与使用
  • Windows 平台
  • Linux 平台
  • Unix 平台

注意：

  • 运行 SuperMap iServer 产品，只需要配置 SuperMap iServer 许可即可。
  • 在安装许可驱动后，建议及时登录许可驱动管理页面，开启许可驱动管理器的安全防护。
