机器学习服务实现AIGIS能力，提供了面向AI领域的多种算法支持，如地物分类、目标检测等。 本节将为您介绍如何配置机器学习服务。

注意：机器学习服务暂不支持 Web 许可，请您参照相关步骤配置普通许可或云许可。

使用二元分类、地物分类、目标检测、场景分类、对象提取等深度学习功能时，才进行下述环境配置，进行聚类、分类、回归相关机器学习功能时无需配置。

  1. 配置Python运行环境

      机器学习服务需要依赖Python运行环境的支持。iServer目前支持配置Python运行环境的方式有：

  • 从官网技术资源中心的“组件 GIS”列表下下载Python运行环境包（supermap-iobjectspy-env-cpu.zip），解压出来是一个conda文件夹，将文件夹放置于【SuperMap iServer 产品目录】/support/python下。
  1. 配置机器学习资源包
  2. 从官网技术资源中心的“组件 GIS”列表下下载机器学习资源包（supermap-iobjectspy-resources-ml.zip），解压出来是一个resources-ml文件夹，放置于【SuperMap iServer 产品目录】/support/python下。该资源包提供有示例数据及示例模型可供参考。
  1. 安装系统库

在 Linux 系统中，除进行以上配置外还需要安装 libsm.so.6 系统库，以下以 Ubuntu 为例介绍 libsm.so.6 的安装。

如计算机可以连接互联网，直接执行以下命令安装 libsm.so.6：

apt-get install libsm6

如不能连接网络，请先于联网环境下从 https://pkgs.org 下载对应系统和位数的安装包，再进行安装。
