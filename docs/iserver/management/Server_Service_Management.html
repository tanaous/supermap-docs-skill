SuperMap iServer 目前提供的 GIS 功能有地图功能、数据功能、分析功能、三维功能等，这些功能以 iServer REST 服务、WMS 服务、WMTS 服务、WFS 服务等形式发布出来供客户端用户使用。详细请参见 SuperMap iServer 服务介绍。

SuperMap iServer 发布 GIS 服务的来源有 SuperMap 工作空间、远程 WMS 服务源、远程 WMTS 服务源、远程 WFS 服务源、远程 iServer REST 服务源等，SuperMap iServer 可以将这些来源发布为各种 SuperMap iServer 支持的 GIS 服务类型。

SuperMap iServer 提供了服务管理的功能，支持快速地发布 GIS 服务和对现有 GIS 服务进行修订。在首次启动 SuperMap iServer 后，访问 URI（本机）：http://localhost:8090/iserver/admin-ui/home/，创建用户账户，就可以进入 SuperMap iServer Web Manager 的登录界面，输入创建的用户名和密码即可进入服务管理页面。

注意：只有首次启动服务时才会进入创建管理员账户界面，以后每次都直接进入 iServer Web Manager 界面，输入第一次创建的用户名和密码即可。
