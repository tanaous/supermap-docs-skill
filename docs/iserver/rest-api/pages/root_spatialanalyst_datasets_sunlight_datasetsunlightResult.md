# datasetSunlightResult

## URI
<[sunlight_uri](sunlight.htm#URI)>/{datasetsunlightResultID}[.<format>]
## 支持的方法
[GET](#GET 请求)、[HEAD](#HEAD 请求)
## 父资源
[sunlight](sunlight.htm)
## 介绍
数据集获取日照分析查询结果资源。
支持的方法：
- [GET](#GET 请求)：返回日照分析查询结果资源。
- [HEAD](#HEAD 请求)：检查 datasetSunlightResult 
资源是否存在，或权限是否可以访问 datasetSunlightResult 资源。
支持的表述格式：RJSON、JSON、HTML、XML。

| 字段 | 类型 | 说明 |

| sunlightpointresults | object | 日照分析点集的属性信息，包括日照点的日照分析、日照时间、点的位置。 |
| geometries | [Geometry3D](../../../../../iServerJavadoc/com/supermap/services/components/commontypes/Geometry3D.html) | 日照分析查询结果模型数组。 |
| succeed | boolean | 日照分析查询是否成功；成功则返回 true，否则返回 false。 |
| message | String | 日照分析查询过程中产生的相关信息。 |
