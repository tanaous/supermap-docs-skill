产品版本介绍

面对应用需求日益细化的 GIS 市场，SuperMap iServer 提供了适用于不同应用类型的系列产品，分别需要不同的运行许可。不同的版本具有不同的访问和管理权限，用户可以根据具体的应用需求选择最适合的 SuperMap iServer 产品，以求达到平台与项目的最优组合。

1. 基础版——简单 GIS 浏览编辑需求，业务快速可视化的首选

针对一些简单的业务可视化需求，SuperMap iServer 提供了适用于文件型数据快速发布、浏览与编辑的基础版产品。

该产品提供了快捷便利的 WebGIS 能力，如地图的发布、浏览、查询与编辑，三维基础能力等。

2. 标准版——数据量大、性能要求高的大中型门户网站建设者的选择

考虑到一些应用系统涉及多源数据且数据量庞大、对性能要求较高， SuperMap iServer 提供了适用于该类型单位的标准版产品。在基础版的基础上，该版本不限制数据来源，支持服务聚合、边缘节点管理，并提供数据目录服务、地址匹配服务、处理自动化服务、单机/多机分布式切图服务和集群服务。

3. 高级版——数据量大、数据源多、性能要求高的大型门户网站建设者的选择

SuperMap iServer 高级版在标准版的基础上，增加了数据流服务、数据科学服务和三方服务分发的功能，内置知识图谱服务模块和视频服务扩展模块，以满足多样性的用户需求。

4. 顶级版——数据量大、对资源弹性有要求的大型门户网站建设者的最佳选择

SuperMap iServer 11i(2024) 开始在高级版的基础上，针对云原生环境提供了顶级版产品。顶级版内置空间分析服务模块、网络分析服务模块和分布式分析服务模块；增加了 AI 能力，内置 GIS 智能体服务，提供自然语言调用 GIS 工具能力；增加 Serverless GIS 能力，即空间分析服务、处理自动化服务、分布式切图服务、Web 打印服务、几何服务和瓦片分发服务以 Serverless 方式运行，此时需搭配云套件顶级版使用。

顶级版只提供按核许可，暂不支持云许可。

各版本许可模块对比

SuperMap iServer 10i 前提供的许可模块均为不限核模块（亦可称为“按套许可”，与 iServer 支持的另一种“按核许可“相反），这种模块不会对 iServer 所在机器的 CPU 逻辑核数进行限制。

SuperMap iServer 从 10i 开始新增按核许可模块，这种许可模块可以对 iServer 所在机器的 CPU 逻辑核数进行控制，iServer 在连接到按核许可模块后，可从许可中心读到当前所在机器的 CPU 逻辑核数，并进行核数校验，要求当前机器的 CPU 逻辑核数小于或等于许可中的核心数。限核模块的核数分别为16核、32核、附加2核，其中附加2核的模块用于扩展其他的限核模块，不能单独使用。

三维服务扩展模块

标准版、高级版和顶级版支持选配。可以实现三维数据发布和浏览、三维空间分析，以及支持各终端的服务访问。

  • 支持三维场景数据发布功能：影像数据、地形数据、KML 数据、模型数据、矢量数据、二维地图、带有符号（Symbols）的三维场景等
  • 提供三维安全机制，对数据发布、数据传输进行安全控制
  • 为提升前端浏览体验，在客户端提供了批量下载数据的功能
  • 支持动态更新，即工作空间有更新时，系统自动更新三维场景、风格、图层
  • 支持数据加载与浏览、图层设置、对象操作、GPU空间查询、GPU空间分析（其中日照分析、可视域分析、天际线分析如果要输出分析结果体对象，需要使用空间分析服务扩展模块）等功能

更多详细信息，请参考使用三维服务。

分布式分析服务扩展模块

标准版和高级版支持选配，顶级版内置。分布式分析服务采用分布式计算技术，提供了对超大体量空间数据集进行分布式空间分析和数据处理的能力。

  • 支持空间统计分析，包括聚合分析、区域汇总分析、属性汇总分析；支持常规的空间分析包括叠加分析、密度分析、缓冲区分析、要素连接、拓扑检查、矢量裁剪；支持数据处理包括创建网格索引。
  • 内置Spark运行库，通过搭建SuperMap iServer集群，即可快速创建分布式分析环境
  • 可快速集成用户已有的Spark集群，省去了系统迁移的代价
  • 支持多种数据来源。包括SuperMap iServer DataStore中的关系数据，时空数据，大数据文件共享中的关系数据、空间数据库中的数据都可进行分布式分析
  • 分析结果支持多种输出。包括将分析结果输出到本地磁盘，SuperMap iServer DataStore以及空间数据库中。用户可根据业务需要，通过API动态调整分析结果的输出目标
  • 支持并发处理多个分析作业，有效解决了因先提交处理长耗时任务，导致短耗时任务不能快速响应的问题，可大幅提升分布式分析服务的吞吐量

更多详细信息，请参考使用分布式分析服务。

机器学习服务扩展模块

标准版、高级版和顶级版支持选配。提供基于机器学习和深度学习相关的高级分析功能。具体包括聚类分析、回归分析、分类分析等数据分析功能，以及基于影像数据的目标检测、地物分类、对象提取功能，以及基于图片视频数据的场景分类，目标检测，目标追踪等功能等。

空间分析服务扩展模块

标准版和高级版支持选配，顶级版内置。支持基于数据集、几何对象进行缓冲区分析、叠加分析、提取等值线和等值面、泰森多边形、插值分析、空间关系分析、线性分析等。此外，在二维服务的基础上，还支持部分三维空间分析服务如：

  • 提供三维实体数据模型的交、并、差等空间运算
  • 提供三维空间查询能力
  • 提供输出日照分析、可视域分析、天际线分析结果为体对象
  • 提供构建阴影体、构建凸包、平面投影、获取模型边界多边形等功能
  • 提供对三维点、三维线、三维面、三维体对象构建三维缓冲区的能力

等等。

更多详细信息，请参阅空间分析服务REST API说明。

流数据服务(Streaming Service)扩展模块

高级版和顶级版支持选配。基于 Spark 生态体系的 Spark Streaming 流处理技术框架。

  • 支持接入Socket、WebSocket、HTTP、JMS、Kafka等主流实时数据传输协议，支持接收CSV、TXT、JSON、GeoJSON等常用数据格式
  • 提供强大的实时分析处理工具，包括属性过滤器、空间过滤器（地理围栏）和转换器
  • 通过发送器实现实时数据输出，支持输出到 iServer DataStore 实现历史数据存储，支持以消息方式实现输出，支持通过数据流服务以WebSocket的方式向客户端实时发送数据，为客户端 SuperMap iClient 实现实时数据可视化提供技术基础。

更多详细信息，请参考使用流数据服务。

注意事项

1、SuperMap iServer 高级版和顶级版中内置了部分扩展服务，您在购买高级版或顶级版的同时将获得对应扩展模块的模块许可。此外您可以根据需要自行选购其他扩展模块。

2、在 SuperMap iServer 任何版本中进行扩展开发，并部署在 SuperMap iServer 中时，不需要单独购买 SuperMap iObjects Java 的许可。

3、SuperMap iServer 所有版本均支持数据的可读可编辑操作。

4、在SuperMap iServer 中支持 WPS 服务，还需购买空间分析服务或网络分析服务扩展模块。

5、服务节点不能单独部署使用，必须加入服务集群，加入后，自动复制集群主节点的所有功能。

6、三方服务分发是指将多源数据发布为Baidu、Google、ArcGIS REST服务等。

7、卫星影像处理服务扩展模块只提供Web许可，针对云原生环境使用。

8、土地覆盖预训练大模型扩展模块、建筑物提取预训练模型扩展模块、水体提取预训练模型扩展模块、耕地提取预训练模型扩展模块、大棚提取预训练模型扩展模块、林地提取预训练模型扩展模块、云提取预训练模型扩展模块暂只提供普通许可、Web许可，且需要与机器学习服务扩展模块许可结合使用。

9、标准版（限3客户端）和标准版（限10客户端）与标准版许可的功能划分、支持的扩展模块完全一致，但对同时访问的用户数和并发数有所限制。当激活标准版（限3客户端）许可时，最大可支持除本机 IP（localhost 或 127.0.0.1）外的三个 IP 所在客户端能对 iServer 进行总量最大为 20 的并发请求；激活标准版（限10客户端）许可时，最大可支持除本机 IP 外的十个 IP 所在客户端能对 iServer 进行总量最大为 70 的并发请求。当客户端长期不访问 iServer 时将被强制下线，访问会话过期时间配置方法详见：配置客户端会话过期时间。此外，同一台机器上只能启动并激活一个标准版（限3客户端）或标准版（限10客户端）的 iServer。
