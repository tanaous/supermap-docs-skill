SuperMap iClient3D for WebGL/WebGPU

Overview

SuperMap iClient 3D for WebGL/WebGPU is a no plug-in, cross-browser client product in the SuperMap service GIS architecture system. It is based on the WebGL/WebGPU and is oriented to the development of HTML 5 3D applications. It could quickly build 3D real space applications with rich content, rapid response and smooth experience.

Starting from SuperMap iClient3D 2023, the product supports both WebGL and WebGPU rendering engines, and uses the same set of API interfaces, which facilitates users to quickly switch according to actual needs and reduces migration costs.
