Download Related Products

There are several related peripheral products about SuperMap iClient3D for WebGL/WebGPU.

  • S3M

    • S3M (Spatial 3D Model) is an open and extensible spatial 3D model tile data format that provides specific data format specifications for the transmission, exchange, and sharing of massive multi-source heterogeneous spatial 3D model data across different terminals.

  • Cesium_S3MLayer_Plugins

    • Cesium_plugins are plugins designed for CesiumJS, and based on these, it is possible to efficiently and quickly implement some of the features of the iClient3D for Cesium product within CesiumJS.

  • iClient3D-Vue

    • Using the Vue framework to encapsulate the SuperMap iClient3D for WebGL product package, providing common functions of 3D applications in the form of components, making it lightweight and efficient.

  • SuperMap-iEarth

    • SuperMap iEarth is a 3D online application developed based on SuperMap iClient3D for WebGL. With the support of multi-source data loading, 3D spatial analysis, and various visualization effects, users can browse and apply 3D scenes in the browser through SuperMap iEarth.
