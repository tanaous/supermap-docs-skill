  <p class="myNormal">SuperMap iPortal 2026 包含以下主要组成部分：</p>
  <h3>前端资源门户</h3>
  <p class="myNormal">SuperMap iPortal 提供了一个前端资源门户，主要面向普通用户，提供地图、服务、场景、数据、项目、洞察、大屏、Notebook 、应用等门户资源的整合、查找与共享能力，并且提供了一个面向普通用户的个人信息管理模块——“<a class="contentpagehyperlink" href="../iportal_usage/user_center.htm">个人中心</a>”，支持编辑个人信息、设置主题风格、添加与管理密钥等。</p>
  <h3 class="myNormal">超图门户AI助理 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了<a class="contentpagehyperlink" href="https://help.supermap.com/iPortalAIAssistant/zh/index.htm" target="_blank">超图门户AI助理</a>（SuperMap iPortal AI Assistant）WebApp，提供全新的GIS交互形式，由常见的复杂菜单和图形界面交互，转变为了简洁和低门槛的问答式交互。通过自然语言交互提出请求，自主调用相关空间智能软件工具完成复杂任务，带来全新的智能交互体验。</p>
  <h3>应用构建器 WebApp（Beta）</h3>
  <p class="myNormal">SuperMap iPortal 提供了一个用于在线构建地图应用的 WebApp——<a href="../iportal_usage/builderX/introduction.htm">应用构建器</a>（BuilderX），快速定制专业的交互式网页地图应用与全屏地图应用</p>
  <h3>数据上图 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了一个用于在线制图的 WebApp ——<a class="contentpagehyperlink" href="../iportal_usage/add_resource/buildmap.htm">数据上图</a>（DataViz），方便用户在线制作与协同分享地图资源。</p>
  <h3>地图工作室 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了一款全新的在线地图应用 WebApp ——<a class="contentpagehyperlink" href="../iportal_usage/mapstudio/ms_overview.htm">地图工作室</a>（MapStudio），支持更加灵活的地图风格配置，助力用户创造更绚丽的地图应用。</p>
  <h3>三维地球 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了<a class="contentpagehyperlink" href="../iportal_usage/scene_usage/createSceceWithEarthApp.htm">三维地球</a> （iEarth）WebApp，实现在线制作三维场景，保存和分享三维场景，进行三维空间分析、量算、裁剪运算等。</p>
  <h3>地图大屏 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了一个开箱即用的<a class="contentpagehyperlink" href="../iportal_usage/mapdashboard/mapdashboard.htm">地图大屏</a> （MapDashboard） WebApp，帮助用户通过图形化的界面，交互式的操作，零门槛轻松搭建各类二三维地图应用。</p>
  <h3 class="myNormal">数据洞察 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了<a class="contentpagehyperlink" href="../iportal_usage/iDataInsights/iDataInsights.htm">数据洞察</a>（DataInsights）扩展模块，可实现对地理数据的可视化表达和空间分析，为用户的业务问题提供帮助，为通用问题寻找分析模型，帮助用户挖掘地理数据价值。</p>
  <h3 class="myNormal">处理自动化建模 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了<a class="contentpagehyperlink" href="../iportal_usage/gpamodelbuilder/gpamodel.htm">处理自动化建模</a>（GPAModelBuilder）WebApp，用简单的拖拽方式快速搭建业务流程模型，助力实现复杂数据处理与分析过程的自动化。</p>
  <h3 class="myNormal">三维地理设计 WebApp</h3>
  <p class="myNormal">SuperMap iPortal 提供了<a class="contentpagehyperlink" href="../iportal_usage/3dDesign/3dDesign.htm">三维地理设计</a>（iDesigner3D）扩展模块，具备基于真实地理环境的在线快速建模和多人交互编辑能力，赋能城市设计、数字交通等行业。</p>
  <h3 class="myNormal">后端管理门户</h3>
  <p class="myNormal">SuperMap iPortal 提供了一个后端管理门户，主要面向管理员提供以下功能：</p>
  <ul class="SymbolList1" style="list-style: disc;">
    <li class="SymbolList1"><span style="font-weight: bold;">门户资源管理</span>：管理员可以集中管理地图、服务、场景、数据、项目、应用、Notebook、群组等门户资源，还可以审核普通用户注册的服务，一键批量注册服务等。</li>
    <li class="SymbolList1"><span style="font-weight: bold;">门户站点配置</span>：管理员可以根据自身业务需求，对门户站点进行配置，如：站点定制、服务代理配置、默认底图配置、内置 WebApp 配置、默认缩略图配置、自定义目录配置等。</li>
    <li class="SymbolList1"><span style="font-weight: bold;">门户监控</span>：管理员可以采取多种措施全面监控门户运行，如：查看日志、服务访问监控、多 GIS 服务器运行监控、异常监控等。</li>
    <li class="SymbolList1"><span style="font-weight: bold;">门户统计</span>：后端管理门户提供了门户统计 Dashboard，方便管理员随时直观了解门户的整体情况、用户的变化情况、资源的使用情况等。</li>
    <li class="SymbolList1"><span style="font-weight: bold;">服务托管</span>： 后端管理门户提供了服务托管能力，支持将多个 SuperMap iServer 服务器节点添加到门户中作为托管服务器，便于用户将上传至门户中的数据随机分散到这些托管的服务器节点中发布服务并自动注册到 iPortal 门户中。</li>
    <li class="SymbolList1" style="font-weight: bold;">用户管理：<span style="font-weight: normal;">管理员可以对内置的账号系统进行集中管理与配置，如：用户的增删、基于角色的访问控制、门户组织结构配置等，还可以集成第三方账号系统，与 iPortal 内的角色进行关联。</span></li>
    <li class="SymbolList1" style="font-weight: bold;"><span>新闻管理：</span><span style="font-weight: normal;">后端管理门户提供了新闻发布能力，支持在线编辑、发布新闻，便于管理员在门户内的信息传达。</span></li>
    <li class="SymbolList1" style="font-weight: bold;">意见反馈审核：<span style="font-weight: normal;">管理员可以查看、审核、回复门户中其他用户提交的意见反馈，通过意见反馈更好地管理门户。</span></li>
  </ul>
  <h3>Java SDK</h3>
  <p class="myNormal">SuperMap iPortal 是可定制的 GIS 门户软件平台，自带了 SDK（Software Development Kit，软件开发工具包），您可以方便的对 SuperMap iPortal 进行扩展，以满足不同业务的特殊需求，使 SuperMap iPortal 门户系统与用户业务业务系统更好的集成。</p>
  <h3>JavaScript SDK</h3>
  <p class="myNormal">SuperMap iClient JavaScript 提供了对接 SuperMap iPortal 中地图、数据、服务等资源的 API，帮助您在自己的 Web 应用中获取或更新 iPortal 中的资源。具体请参见：<a class="contentpagehyperlink" href="../../GettingStarted/iServer_iClient_Start/JS_API_iPortal.htm">使用 iClient JavaScript API 浏览 iPortal 地图</a>。</p>
  <p class="myNormal" style="font-weight: bold;">注意：SuperMap iPortal 8C SP1 及更新的版本，支持独立部署，不再内置 SuperMap iServer 应用服务器。</p>
