  <h3>顶部导航栏</h3>
  <p>顶部导航栏提供了新手引导、语言切换、帮助文档与账号切换的功能入口。</p>
  <h3>侧边栏</h3>
  <p>侧边栏提供了会话、指令中心和历史记录的功能入口。</p>
  <ul class="Disc">
    <li>指令中心：包含了AI助理所有内置工具的介绍说明，并为每个工具提供了一条指令样本，点击复制到对话框中，即可快速体验；</li>
    <li>历史记录：包含了当前用户所有的会话记录，支持搜索、查看、编辑与删除。</li>
    <li>设置：包含了AI助理的常用设置项，包括模型设置、知识库设置与对话设置。</li>
    <li>工作台（仅管理员可见）：面向管理员，用于工具和知识库的管理。</li>
  </ul>
  <h3>会话区</h3>
  <p>会话区支持通过自然语言的方式与AI助理进行交互对话，AI助理在理解您的请求后，能够自主调用相关工具完成相应的任务，并给出对话建议。</p>
  <h3>展示区</h3>
  <p>展示区用于展示GIS工具调用后返回的结果，如属性表、地图、三维场景、遥感影像等；在未调用工具时，该区域也用于指令中心的内容展示。</p>
