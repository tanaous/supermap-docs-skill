<h3><a id="zip_包_for_Windows"></a>zip 包 for Windows</h3>
<p class="myNormal">SuperMap iPortal 针对 Windows x64 系统提供了完整 zip 包和 
 deploy 版的 zip 分发包，其中 deploy 分发包不含示范代码、帮助文档及客户端开发工具包。</p>
<p class="myNormal">SuperMap iPortal 的完整 zip 包产品在正常解压缩完成后，目标计算机应具有以下的内容：</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">bin</span> 
 文件夹：启动、停止 Tomcat 服务，以及其他相关的脚本文件，Windows 下从 startup.bat 启动服务。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">conf</span> 
 文件夹：存放 Tomcat 的配置文件。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">database</span> 
 文件夹：存放SuperMap iPortal 内置的ElasticSearch数据库。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">docs</span> 
 文件夹：存放 SuperMap iPortal 的帮助文档和电子书。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">iClient</span> 
 文件夹：存放 SuperMap iPortal 的客户端，包括各个开发平台上的工具库、示范代码等内容，如 forJavaScript、forAndroid、foriOS 
 等。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">lib</span> 
 文件夹：存放 Tomcat 所需的 jar 文件。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">licenses</span> 
 文件夹：存放 SuperMap iPortal 用到的开源库的许可协议和声明。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">logs</span> 
 文件夹：存放 SuperMap iPortal 和 Tomcat 启动、运行过程中的日志信息。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">samples</span> 
 文件夹：存放 SuperMap iPortal 的示范程序。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">support</span> 
 文件夹：存放 JRE、许可配置工具及产品升级脚本工具。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">temp</span> 
 文件夹：Tomcat 运行过程中的存放临时文件的位置。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">webapps</span> 
 文件夹：SuperMap iPortal 部署的文件夹，里面的 iPortal 文件夹下存放有 SuperMap iPortal 
 依赖的 jar 文件、配置文件等信息。</p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">license_*.txt</span> 
 文件：SuperMap iPortal 用户许可协议文件。</p>
<p class="myNormal" style="font-weight: bold;">readme_*.txt <span class="hcp4" style="font-weight: normal;">文件：与安装产品并启动服务相关的自述文件，用户可据此安装并启动服务。</span></p>
<p class="myNormal" style="font-weight: bold;">SuperMap_iPortal_10i(2021)_Readme_Windows_*.pdf 
 <span class="hcp4" style="font-weight: normal;">文件：SuperMap</span> <span style="font-weight: normal;">iPortal</span> <span class="hcp4" style="font-weight: normal;">的自述文件。</span></p>
<p class="myNormal"><span class="hcp3" style="font-weight: bold;">BUILD</span><span class="hcp3">_*</span> 文件：SuperMap iPortal 的版本（由*的内容标识）。</p>
<p class="myNormal">其他文件为 Tomcat 的许可、声明等信息。</p>
<h3><a id="tar_包_for_Linux"></a>tar 包 for Linux</h3>
<p class="myNormal">SuperMap iPortal 的64 bit 产品（for Linux）在执行 tar -zxvf 
 *.tar.gz 命令正常解压完成后，得到 supermap_iportal_*_linux64 文件夹，其具体内容请参见 <a class="contentpagehyperlink" href="#zip_包_for_Windows">zip 
 包 for Windows</a>，其中自述文件名为 SuperMap_iPortal_10i(2021)_Readme_Linux_CHS.pdf。</p>
<p class="myNormal">SuperMap iPortal 不仅提供了完整 tar 分发包，还提供了轻量版地 deploy 分发包供用户使用。其中 
 deploy 分发包不含示范代码、帮助文档及客户端开发工具包。</p>
