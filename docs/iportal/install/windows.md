  <p class="myNormal">SuperMap iPortal 2026 目前提供了 64 位的解压缩包（以.zip 为扩展名）这种形式的分发包供用户在 Windows 系统上使用。</p>
  <h3><a id="安装配置_可选_"></a>安装前配置（可选）</h3>
  <p class="myNormal">SuperMap iPortal 默认无需任何安装配置即可满足您试用的需求。如果您将 iPortal 用在实际的生产环境中，建议您根据自身业务需要，进行如下可选的安装配置。</p>
  <ul class="referenceList">
    <li class="referenceList"><a class="contentpagehyperlink" href="database_config.htm">数据库配置</a>：SuperMap iPortal 的门户数据默认存储在 SQLite 数据库中，无需安装和配置，但是针对大数据量、高并发的情况，建议您配置 MySQL/Oracle/PostgreSQL 数据库。</li>
    <li class="referenceList"><a class="contentpagehyperlink" href="accountEnvironment.htm">账户初始化配置</a>：SuperMap iPortal支持通过环境变量配置，初始化管理员账户，启动服务后，无需进入“配置向导”页面创建账户，即可直接进入门户首页，使用初始化的管理员账户登录。</li>
  </ul>
  <h3><a id="产品安装"></a>产品安装</h3>
  <p class="myNormal">建议 SuperMap iPortal 的安装路径为英文目录，且需注意路径中不能带有空格符。解压产品压缩包后，在【SuperMap iPortal 产品目录】/bin 目录下，您将看到启动 iPortal 服务的批处理文件 startup.bat，直接双击运行即可。</p>
  <p class="myNormal">如果 SuperMap iPortal 的产品目录在操作系统盘（C 盘）下，会因操作系统的权限控制无法写入 log 文件而导致服务启动失败，此时需要以管理员身份来运行 startup.bat。具体操作时，以管理员身份打开 <span style="font-size: 12px; 
	 orphans: auto; text-indent: 24px; widows: auto; -webkit-text-stroke-width: 0px; 
	 background-color: #ffffff; display: inline	 ! important ; float: none;">cmd.exe（管理员：命令提示符），切换路径到</span>【SuperMap iPortal 产品目录】/bin，运行 startup.bat 如下：</p>
  <p class="myCodeP">C:\WINDOWS\system32&gt;cd /d C:\Windows\supermap-iportal-10.1.1-win64-zip\bin</p>
  <p class="myCodeP">C:\Windows\supermap-iportal-10.1.1-win64-zip\bin&gt;startup.bat</p>
  <h3><a id="安装后配置_可选_"></a>安装后配置（可选）</h3>
  <p class="myNormal"><span class="hcp2">仅当用户不希望使用 SuperMap iPortal 2026 自带的 JRE（Java Runtime Environment）时，才需要手工配置</span>。SuperMap iPortal 2025默认使用包中自带的 JRE（位于 %SuperMap  iPortal _HOME%/support 目录下）。</p>
  <p class="myNormal"><span class="hcp2">手工配置前</span>，需要将%SuperMap iPortal _HOME%/support 目录下的 jre 目录重命名或删除。</p>
  <p class="myNormal">JRE（1.8 或以上版本）设置环境变量如下：</p>
  <ul class="SymbolList1" style="list-style: disc;">
    <li class="SymbolList1">如果安装的是 JDK，则设置 JAVA_HOME 为 JDK 的目录。</li>
    <li class="SymbolList1">如果安装的是 JRE，则设置 JRE_HOME 为 JRE 的目录。</li>
  </ul>
  <p class="myNormal">高级用户也可通过直接修改 %SuperMap  iPortal_HOME%/bin/setenv.bat 来自定义 JRE。</p>
  <h3><a id="产品卸载"></a>产品卸载</h3>
  <p class="myNormal">停止 iPortal 服务后，直接删除 SuperMap iPortal 2026 解压目录即可。</p>
