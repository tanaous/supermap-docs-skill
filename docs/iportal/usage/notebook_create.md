<p class="myNormal">Notebook 是 SuperMap iPortal 中提供的在线交互式开发环境，支持在线创建、运行 
 Python、Markdown、Raw 等多种脚本，并基于空间数据进行分布式分析、机器学习等工作。需要注意的是，如果要在 Notebook 
 中运行创建的脚本，门户管理员需要先在管理页面<a class="contentpagehyperlink" href="../../iportal_management/addNode.htm#添加数据科学服务器">添加数据科学服务器</a>。</p>
<p class="myNormal">下面将详细介绍 iPortal 中的 Notebook 页面的构成，以及如何创建 Notebook 资源。</p>
<h2>Notebook页面介绍</h2>
<p class="myNormal">在 iPortal 中，未添加数据科学服务器的 Notebook 仅支持登录用户在线创建与保存脚本。门户管理员添加数据科学服务器后，Notebook 
 中的运行功能方可启动，登录用户可以从门户首页导航栏的“资源中心”下拉菜单中点击 &quot;Notebook&quot;，点击右侧的“创建 Notebook”进入 Notebook 
 页面，在线创建、运行 Python 等多种脚本，如下图所示：</p>
<p class="myNormal" style="text-align: center;"><img src="../../../assets/images/iP/iportal_usage/img/notebook.png" alt="" style="border: none;" border="0"/></p>
<ul style="list-style: disc;">
	<li><span style="font-weight: bold;">顶部导航栏</span>：显示、编辑当前 Notebook 
	 的名称，并放置保存、另存为（保存 Notebook 资源后出现）按钮。例如，当前 Notebook 的名称为 Untitled（未保存前的默认名称），您可以在导航栏中双击 
	 &quot;Untitled&quot;，就可以为当前的脚本命名了，您也可以在内容添加完成后，点击导航栏中的保存按钮，在弹出的对话框中输入 
	 Notebook 资源的名称。</li>
	<li><span style="font-weight: bold;">常用工具栏</span>：</li>
</ul>
<ul class="SymbolList2">
	<li class="SymbolList2">常用功能按钮：包括插入、剪切、粘贴、复制代码块，运行、停止、重启服务。</li>
	<li class="SymbolList2">编辑器：显示当前编辑器（默认 Code），您也可以在此切换 Markdown、Raw 
	 编辑器。</li>
	<li class="SymbolList2">Kernel：显示当前运行的核（默认连接数据科学服务器中运行的核，此例中为 Python3），并支持重新连接、更改核。</li>
	<li class="SymbolList2" style="color: #000000;">Kernel Idle：指示连接的数据科学服务器的运行状态，正常运行显示为绿色，非正常运行显示为红色。</li>
</ul>
<ul style="list-style: disc;">
	<li><span style="font-weight: bold;">工作空间</span>： 显示用户在 Notebook 中创建的内容，可包括多个单元格。</li>
</ul>
<h2 class="myNormal">创建Notebook资源</h2>
<p class="myNormal">在 iPortal 中创建 Notebook 资源的用户需是“创建者”类型且拥有创建资源权限的角色，如 
 PORTAL_USER（iPortal的用户类型及权限划分，详见：<a class="contentpagehyperlink" href="../../iportal_management/Portal_Security/RolesandPermissions.htm">角色、用户类型和权限</a>）。</p>
<p class="myNormal">下面以 Python3 代码实现 DBSCAN 密度聚类的示例，为您介绍如何在 iPortal 中创建 
 Notebook 资源：</p>
<ol style="list-style: decimal;" class="ListInParagraph">
	<li class="ListInParagraph">
	  <p>在 iPortal 首页，从门户首页导航栏的“资源中心”下拉菜单中点击 
	 &quot;Notebook&quot;，点击右侧的“创建 Notebook”进入 Notebook 页面。确认 Kernel Idle 指示灯显示为绿色，若指示灯长时间显示为红色，则需要查看数据科学服务器的运行状态，并尝试重新连接；</p></li>
	<li class="ListInParagraph"><p>在常用工具栏中，选择编辑器 &quot;Code&quot; ，选择 Kernel（运行的核）为 
	 &quot;Python3&quot; ；</p></li>
	<li class="ListInParagraph"><p>在工作空间输入示例代码，在常用工具栏中点击运行按钮，运行该 Notebook，如下图所示：</p></li>
</ol>
<p style="text-align: center;"><img src="../../../assets/images/iP/iportal_usage/img/notebook1.png" alt="" style="border: none;" border="0"/></p>
<p class="myNormal">运行结果显示在代码块下方，该示例显示为“DBSCAN 密度聚类成功，结果数据集为 taxi_up_webMercator_dbscan_1”，如下图所示：</p>
<p class="myNormal" style="text-align: center;"><img src="../../../assets/images/iP/iportal_usage/img/notebook2.png" alt="" style="border: none;" border="0"/></p>
<ol start="4" style="list-style: decimal;" class="ListInParagraph">
	<li class="ListInParagraph"><p>运行成功后，点击“保存”按钮，输入名称并确认后，即可完成一个 Notebook 
	 资源的创建。</p></li>
	<li class="ListInParagraph"><p>创建成功后，您可以到“资源中心”&gt;“Notebook”页面查看 Notebook 
	 资源列表，还可以对有权限的 Notebook 资源进行编辑、属性编辑、批量删除等操作，同时可以根据需要<a class="contentpagehyperlink" href="../share_resource/shareNotebook.htm">共享资源</a>。此外，iPortal 
	 管理员用户还可以在“管理”&gt;“资源管理”&gt;&quot;Notebook&quot; 页面，管理门户中已添加的所有 Notebook 
	 资源，详见<a class="contentpagehyperlink" href="../../iportal_management/Notebookmg.htm">Notebook管理</a>。</p></li>
</ol>
<p class="myNormal"><span style="font-weight: bold;">注</span>：可保存的 Notebook 
 数量默认无限制，如果管理员设置了<a class="contentpagehyperlink" href="../../../Subject_introduce/Security/config_role/users.htm#更新_WebApps_成果保存数量">创建 
 Notebook 数量的配额</a>，页面上会以进度条的形式展示配额使用情况，当创建 Notebook 的数量达到配额上限后，无法再保存 Notebook，此时您可以联系管理员提升配额或删除一些已创建的 
 Notebook 以释放配额空间。</p>
<p> </p>
