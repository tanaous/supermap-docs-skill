          <div class="RH-LAYOUT-CENTERPANEL-topic-box" id="rh-topic">
  <p class="myNormal">SuperMap iPortal 提供了 BuilderX 应用构建器（Beta） WebApp，它是一款在线地图应用构建工具，无需编码，通过简单拖拽的方式，快速定制专业的交互式地图应用。可创作垂直滚动页面与全屏应用，放置专注于网页搭建的可视化组件，布局灵活自由，助力用户创建独特的现代化地图网页。</p>
  <h2>使用入口</h2>
  <p class="myNormal">登录 iPortal 后，您可通过以下方式访问应用构建器进行应用创作（要求用户是“创建者”类型且拥有创建、更新、删除应用权限的角色，如 PORTAL_USER。iPortal 的用户类型及权限划分，详见：<a href="../../iportal_management/Portal_Security/RolesandPermissions.htm">角色、用户类型和权限</a>）：</p>
  <ul style="list-style: disc;">
    <li class="SymbolList1">在“资源中心” &gt; “应用”页面，点击右侧页面中的“创建应用”，进入应用构建器的创建页面。</li>
    <li class="SymbolList1">在“应用中心”页面，点击左侧侧边栏的“应用构建器”，在右侧欢迎页点击“新建”，进入应用构建器的创建页面。</li>
  </ul>
  <h2>使用流程</h2>
  <p class="myNormal">使用应用构建器的一般流程是：创建应用、添加数据、添加可视化组件、预览、保存与分享。<span>在构建过程中，支持快捷键撤销（Ctrl+Z）、恢复（Ctrl+Y）操作、复制（Ctrl+C）、粘贴（Ctrl+V）操作。</span></p>
  <ul style="list-style: disc;">
    <li class="SymbolList1">创建应用：可创建滚动页面和全屏应用页面，支持基于空白画布、应用构建器提供的布局或模板创建应用。</li>
    <li class="SymbolList1">添加数据：某些可视化组件（如文本组件、文本列表组件）需要关联数据进行使用，可通过数据面板添加、查看与管理组件中涉及到的数据。</li>
    <li class="SymbolList1">添加可视化组件：可通过拖拽的方式向页面中添加布局、网页、地图、场景、图表、基础等类型的组件，并配置组件的样式风格、不同组件之间的联动交互。</li>
    <li class="SymbolList1">预览、保存与分享：对于制作完成的应用，可以预览其显示效果，然后进行保存与分享等操作，以实现应用成果的应用。</li>
  </ul>
  <p class="myNormal">除此之外，应用构建器也具备扩展定制的能力，支持用户自定义组件及其配置项、左侧 Tabs 标签页，顶部工具栏以及支持用户接入自定义数据，请参见：<a href="https://gitee.com/isupermap/builderx-extensions-samples">扩展开发</a>。</p>
