  <p class="myNormal">地图工作室 MapStudio 是一款全新的在线地图应用，支持更加灵活的地图风格配置，助力用户创造更绚丽的地图应用。</p>
  <p class="myNormal">登录 iPortal 后，您可通过以下方式访问地图工作室 WebApp 进行地图创作（要求用户是“创建者”类型且拥有创建、更新、删除地图权限的角色，如PORTAL_USER。iPortal的用户类型及权限划分，详见：<a class="contentpagehyperlink" href="../../iportal_management/Portal_Security/RolesandPermissions.htm">角色、用户类型和权限</a>）：</p>
  <ul style="list-style: disc;">
    <li class="SymbolList1">
      <p>在“资源中心”&gt;“地图”页面，点击右侧“创建地图”，在下拉列表中选择“地图工作室”，即可进入地图工作室页面。</p>
    </li>
    <li class="SymbolList1">
      <p>在“应用中心”页面，点击左侧侧边栏的地图工作室 Logo，在右侧“地图工作室 WebApp”欢迎页点击“新建”，即可进入地图工作室页面。</p>
    </li>
  </ul>
  <p class="myNormal">使用地图工作室的一般流程是：选择底图、添加图层（数据）、配置图层风格、保存与分享。其中底图支持传统的地图底图和纯色底图。选择底图后，可直接添加来源于 iServer REST 地图服务、矢量瓦片服务、或门户中“我的服务”中的地图作为图层，也可上传数据作为图层添加到底图上。已添加的图层支持设置显隐、调整图层显示次序、重命名、移除、放缩至等操作。对选中的图层还可查看属性详情、修改图层显示风格。</p>
  <p class="myNormal">您可以随时调整地图工作室左侧、右侧功能面板的宽度，以适应不同分辨率的显示器，获得更好的制图体验。</p>
  <p class="myNormal">注：在使用地图工作室前，门户管理员可以在“管理”&gt;“应用中心配置”&gt;“地图工作室”页面和“结构化数据”页面，进行<a class="contentpagehyperlink" href="../../iportal_management/Portal_config/mapstudio_config/mapstudio_config.htm">地图工作室配置</a>和<a class="contentpagehyperlink" href="../../iportal_management/structuralData/StructuralDataManagement.htm">结构化数据配置</a>（上传数据要求)。</p>
