          <div class="RH-LAYOUT-CENTERPANEL-topic-box" id="rh-topic">
  <p class="myNormal">SuperMap iPortal 作为 GIS 门户软件平台，提供了地图、服务、场景、数据、洞察、大屏、GPA 模型、3D设计、应用、项目、Notebook等 GIS 资源整合与查找功能，您可以在资源中心访问与浏览整合的各类 GIS 资源，以及使用搜索功能方便、快捷地查找定位需要的 GIS 资源。</p>
  <h2 class="myNormal"><a id="资源浏览"></a>资源浏览</h2>
  <p class="myNormal">在 iPortal 导航栏中点击“资源中心”，可以进入资源中心页面。<span>您可以在此集中查找、访问和管理所有资源。页面顶部的二级导航栏根据资源的所有权和共享范围对资源进行了分类，在二级导航栏中您可以浏览：</span></p>
  <ul class="SymbolList1" style="list-style: disc;">
    <li>公开的资源：展示您可访问的所有公开资源</li>
    <li>分享给我的资源：展示其他用户分享给您的资源</li>
    <li>我的组织资源：展示您所属部门内共享的资源，此标签页仅在 iPortal 开启了<a class="contentpagehyperlink" href="../iportal_management/Portal_Security/organiz_struc_set.htm">组织结构配置</a>后显示</li>
    <li>我的群组资源：展示您加入或您创建的群组内共享的资源</li>
    <li>我的资源：展示您个人创建和拥有的资源</li>
    <li>我的收藏：展示您标记为收藏的资源</li>
  </ul>
  <p class="myNormal"><span>您还可以选择偏好的</span><a class="contentpagehyperlink" href="#更改视图模式">视图模式</a><span>以浏览资源，</span><span>支持列表模式与格网模式两种视图模式，服务资源还支持预览视图，以实现服务的快速上图浏览。</span></p>
  <p class="myNormal"> </p>
  <h2 class="myNormal"><a id="资源过滤"></a>资源过滤<span></span></h2>
  <p class="myNormal"><span>在资源页面，可以通过</span><a class="contentpagehyperlink" href="#按照服务类型分类过滤1">分类过滤</a>、<a class="contentpagehyperlink" href="#按照标签分类过滤">标签过滤</a>、<a class="contentpagehyperlink" href="#featuredFilter">精选过滤</a>、<a class="contentpagehyperlink" href="#favoriteFilter">收藏过滤</a>等功能定位感兴趣的资源，通过<a class="contentpagehyperlink" href="#排序">分类排序</a>功能查找最新、最热的资源。</p>
  <h3 class="myNormal"><a id="按照服务类型分类过滤1"></a>分类过滤</h3>
  <p class="myNormal">分类过滤是指把不同种类的资源按照其来源、类型等进行分类，以便于在资源列表中快速找到某一类的资源，<span>iPortal 支持以下分类过滤方法：</span></p>
  <ul class="SymbolList1" style="list-style: disc;">
    <li class="SymbolList1">按资源类型分类</li>
  </ul>
  <p class="myNormal">iPortal 的服务、项目资源支持按类型进行过滤，在资源页面左侧过滤器的“类型”一栏，可通过选择过滤出不同类型的资源：</p>
  <ul class="SymbolList2" style="list-style: disc;">
    <li class="SymbolList2">服务资源，支持按照服务的类型进行过滤，如：“SuperMap REST”、“ArcGIS REST”、“WMS”、“WFS”、“WMTS”、“WCS”、“WPS”等；</li>
    <li class="SymbolList2">项目资源，支持按照项目的类型进行过滤，如：“Web 应用”、“PC 应用”、“移动端应用”等。</li>
  </ul>
  <ul class="SymbolList1" style="list-style: disc;">
    <li class="SymbolList1">按自定义目录分类</li>
  </ul>
  <p class="myNormal">iPortal 支持管理员进行<a class="contentpagehyperlink" href="../iportal_management/Portal_config/customDirectorySetting.htm">自定义目录配置</a>，以满足实际应用需要。<span>管理员进行自定义目录配置后，在相应资源类型页面的左侧过滤器中会依次显示配置的自定义目录类别，每个目录类别会列出具体的目录结构。如果您想调整目录分类、将新添加的资源进行归类，请联系 iPortal 管理员。</span></p>
  <h3 class="myNormal"><a id="按照标签分类过滤"></a><span style="font-weight: bold;">标签过滤</span></h3>
  <p class="myNormal"><span>为资源添加标签可在资源列表中更容易地识别资源。在资源页面中，可以通过点击左侧过滤器中某一个标签。即可过滤出所有带有该标签的资源</span><span>。</span></p>
  <h3><a id="featuredFilter"></a>精选过滤</h3>
  <p class="myNormal">门户管理员可通过<a class="contentpagehyperlink" href="../iportal_management/resourceManag.htm">资源管理</a>功能将门户内的资源标记为精选资源。精选资源在展示时会携带精选标记。您在资源中心浏览资源时，可通过精选标记识别精选资源，也可直接打开“精选面板”开关，查看门户内的精选资源。</p>
  <h3 class="myNormal"><a id="favoriteFilter"></a>收藏过滤</h3>
  <p class="myNormal">用户可通过<a class="contentpagehyperlink" href="../iportal_management/resourceManag.htm">资源管理</a>将门户内的资源设置为收藏资源。您在资源中心浏览资源时，可通过我的收藏查看收藏资源。</p>
  <h3 class="myNormal"><a id="排序"></a>分类排序</h3>
  <p class="myNormal">分类排序是将资源按照一定的特征进行排序，方便用户更清晰、直观地找到目标资源，在资源中心页面，点击页面右侧的排序方式下拉列表，可以选择以资源名称、修改时间、<span>收藏量、访问数、下载数</span>等方式对资源进行排序，以及切换升序或降序排列，以快速找到门户中最新/最热的资源。</p>
  <p class="myNormal"> </p>
  <h2 class="myNormal">资源检索</h2>
  <p class="myNormal">iPortal 支持以下两种方式进行资源的检索，以协助您方便、快捷地查找定位需要的 GIS 资源。</p>
  <h3><a id="模糊查询"></a><span style="font-weight: bold;">使用资源页面搜索框检索</span></h3>
  <p class="myNormal">在资源页面左侧的搜索框中输入关键字符，如“china”，按回车键或点击搜索按钮即会列出符合条件的搜索结果，支持<span>使用资源名称、资源标签、资源描述、自定义的元信息中的关键</span><span>字</span>来检索。您可以配合使用<a class="contentpagehyperlink" href="#资源浏览">资源浏览</a>中的多种过滤方式，来缩小搜索结果范围以进行资源的定位。</p>
  <h3><a id="全局搜索"></a>全局搜索</h3>
  <p class="myNormal">全局搜索可以根据搜索条件，在门户所有类型资源中检索出与搜索条件相符的资源，而无需在每类资源页面中逐一进行搜索，<span>全局搜索支持以下两种方式</span>：</p>
  <ul class="SymbolList1" style="list-style: disc;">
    <li class="SymbolList1">基于关键字符进行检索：在 iPortal 导航栏中的全局搜索框中输入关键字符，如“world”，按回车键或点击搜索按钮即可，iPortal 会自动匹配名称、描述信息及自定义的元信息与关键字匹配的资源。</li>
  </ul>
  <ul style="list-style: disc;">
    <li class="SymbolList1">基于地理范围的检索：通过设置地理范围，对带有地理坐标信息的资源与设置的范围进行匹配，从而筛选出位于指定范围内的资源，iPortal 提供以下两种方式设置地理范围：</li>
    <li class="SymbolList2">在“地图范围”输入框中输入左下角及右上角经纬度坐标，设置一个以此两点为顶点的矩形区域；</li>
    <li class="SymbolList2">单击画框设置图标<img alt="" border="0" height="18" src="../../assets/images/iP/iportal_usage/img/drawFrame.png" style="border: none;" width="18"/>，在弹出的地图窗口中，点击 <img alt="" border="0" src="../../assets/images/iP/iportal_usage/img/draw.png" style="border: none;"/> 按钮，以拖拽的方式绘制一个矩形区域，然后点击”应用“完成地理范围的设置。</li>
  </ul>
  <p class="myNormal">您可以将关键字符与地理范围两种条件搭配使用，以实现更精细与准确的搜索。同时，全局搜索的结果也支持标签过滤和分类排序，方便您清晰直观地找到目标资源。</p>
  <p class="myNormal">iPortal 全局搜索能力基于 ElasticSearch 搜索服务器实现，并支持配置全局搜索的服务地址以及全局搜索服务器的参数，关于配置方式的详细说明，请参考：<a class="contentpagehyperlink" href="../iportal_management/Portal_config/searchServiceConfig.htm">搜索服务配置</a>。</p>
  <p class="myNormal"> </p>
  <h2 class="myNormal">资源访问</h2>
  <p class="myNormal">在检索与定位到所需的资源后，您可以访问资源的详情。对于具有查看资源的服务，可以预览该服务的可视化效果；对于具有查看权限的地图、场景、洞察、大屏等类型的资源，可以进一步在 WebApps 中查看，而对于无查看权限的资源，可以向资源所有者进行<a class="contentpagehyperlink" href="apply%26authorizeResource.htm">申请授权</a>。</p>
  <h3 class="myNormal">查看资源详情</h3>
  <p class="myNormal">点击资源的缩略图或资源条目的“详情”按钮，可以查看资源的详细信息，资源详情包括元信息和共享设置：</p>
  <ul style="list-style: disc;">
    <li class="SymbolList1">元信息：资源的元信息包括资源的名称、缩略图、标签、作者、更新时间、访问/下载数、资源描述信息等。此外，某些类型的资源还拥有一些特有的元信息项：</li>
    <li class="SymbolList2">地图：地图的坐标系信息、中心点、地图范围以及地图的图层列表</li>
    <li class="SymbolList2">服务：服务的地址（iPortal 默认启用了<a class="contentpagehyperlink" href="../iportal_management/Portal_config/serviceProxy_config/Service_Proxy_Config.htm">服务代理功能</a>，将显示服务的代理地址）、类型、服务参照系、服务标识、服务范围以及服务提供者信息</li>
    <li class="SymbolList2">数据：数据的类型、数据范围、坐标系、数据提供者、文件大小等。在数据发布了托管服务后，还包括数据已发布的服务信息</li>
    <li class="SymbolList1">共享设置：可查看并修改当前资源的共享范围，仅资源所有者与 iPortal 管理员有权限进行操作，具体请参考：<a class="contentpagehyperlink" href="share_resource/share.htm">共享资源</a>。</li>
  </ul>
  <h3 class="myNormal"><a id="预览服务"></a>预览服务</h3>
  <p class="myNormal">对于具有查看权限的服务资源，点击服务条目的“服务预览”按钮，可以查看服务的可视化效果，iPortal 提供了以下几种预览模式：</p>
  <ul class="SymbolList1" style="list-style: disc;">
    <li class="SymbolList1">预览视图：通过 iPortal 资源中心视图模式中的预览视图查看服务</li>
    <li class="SymbolList1">上图预览：通过 iPortal 数据上图 WebApp 查看服务</li>
    <li class="SymbolList1">iClient 预览：通过可视化交互开发平台 SuperMap iClient JavaScript 查看服务</li>
    <li class="SymbolList1">自定义预览：通过服务所有者指定的地址查看服务（需服务所有者设置自定义预览地址），通过自定义预览可以查看无法在 iPortal 中进行可视化的服务，如 ArcGIS REST 服务等</li>
  </ul>
  <h3 class="myNormal">在 WebApp 中查看资源</h3>
  <p class="myNormal">对于具有可查看及以上权限的地图、场景、洞察、大屏、应用等资源，可以通过 iPortal 内置的 WebApps 进行查看，具体请参考：<a class="contentpagehyperlink" href="view_resource/viewResourceWithWebApp.htm">在 WebApp 中查看资源</a>。</p>
