  <p class="myNormal" style="color: #000000;">作为 GIS 门户软件平台，SuperMap iPortal 支持整合 SuperMap GIS 格式、常见 GIS 格式以及含地理位置坐标的 Excel、CSV 等传统数据，也支持接入 HDFS 和数据库数据。除 GIS 数据外，也以 REST API 的方式支持添加音视频、图片等非 GIS 数据。添加到门户中的 GIS 数据可在内置 WebApps 中直接采用地图或多种图表进行可视化表达、深度分析。也可发布成服务后，以服务的形式供各内置 WebApps 使用或供其他 Web GIS 应用系统调用。下面分别详细介绍在门户中添加数据的两种方式：上传数据和注册数据，以及如何使用 iPortal 中添加的数据。注意：若管理员开启了<a class="contentpagehyperlink" href="../../iportal_management/datamanage.htm#DataAudit">数据审核</a>功能，新添加到门户中的数据审核通过后，才能在门户中搜索与使用。</p>
  <h2><a name="uploadData" id="uploadData"></a>上传数据</h2>
  <p class="myNormal" style="color: #000000;">在 iPortal 中上传数据要求用户是“创建者”类型且拥有上传数据和发布服务权限的角色，如 DATA_CENTER（iPortal 的用户类型及权限划分，详见：<a class="contentpagehyperlink" href="../../iportal_management/Portal_Security/RolesandPermissions.htm">角色、用户类型和权限</a>）。此外，需要管理员进行<a class="contentpagehyperlink" href="../../iportal_management/Services_hosting/Storage_Config.htm">托管数据存储配置</a>，并<a class="contentpagehyperlink" href="../../iportal_management/Services_hosting/Specify_HostedServer.htm">指定托管服务器</a>用于发布服务。</p>
  <p class="myNormal" style="color: #000000;">iPortal 支持配置二进制和关系型的托管数据存储，默认使用二进制存储。若管理员同时配置了关系型存储，系统将优先使用关系型存储，同时也会在二进制存储中备份上传的数据。但也有一些数据如缓存切片仅支持存储在二进制中。上传数据支持的数据类型、发布服务类型及下载格式如表1所示。</p>

  <p class="myNormal"> </p>
  <p class="myNormal">若您有管理非 GIS 数据，如音视频等资源的需求，可通过 REST API 的形式上传 ，详情请参阅：<a class="contentpagehyperlink" href="../../../mergedProjects/SuperMapiServerRESTAPI/Portal/myContent/myDatas/myDatas.htm#POST_请求">myDatas 资源的 POST 请求</a>。</p>
  <p class="myNormal">此外，iPortal 对上传到门户中的数据容量有大小限制，默认您可上传的最大容量为2048MB，管理员可修改此项配置，详见：<a class="contentpagehyperlink" href="../../iportal_management/Portal_config/data_storage_config.htm#上传数据容量配置">上传数据容量配置</a>。</p>
  <p class="myNormal"> </p>
  <p class="myNormal" style="font-weight: bold;">上传步骤：</p>
  <ol class="ListInParagraph" style="list-style:decimal;">
    <li class="ListInParagraph">在 iPortal 首页，点击“资源中心”&gt;“数据”进入数据资源页面，右侧点击“上传数据”（需登录）进入上传数据页面；</li>
    <li class="ListInParagraph">选择“数据类型”，指定要上传的“数据文件”，设置“数据名称”及“标签”后，点击“上传”，完成数据上传；点击“上传并发布”，可将数据上传并发布为服务<span>，若使用共享存储发布服务，请参考：</span><a href="../../iportal_management/Services_hosting/Storage_Config.htm#使用共享存储发布服务">使用共享存储发布服务</a>；</li>
    <li class="ListInParagraph">若选择的是“上传并发布”，将进入发布服务页面，勾选要发布的服务类型，点击“下一步”，进行服务参数配置；</li>
    <li class="ListInParagraph">当选择发布为交通网络分析服务时，参数说明请参考：<a class="contentpagehyperlink" href="../../../Server_Service_Management/GISService_Config/Service_Config/Config_Provider/ProviderParameter/TransportationProviderParam.htm" style="color: rgb(6, 105, 178); cursor: pointer; font-family: 微软雅黑, Verdana, Arial, Helvetica, sans-serif, 宋体; 
	 font-size: 13px; font-style: normal; font-variant-ligatures: normal; 
	 font-variant-caps: normal; font-weight: 400; letter-spacing: normal; 
	 orphans: 2; text-align: left; text-indent: 0px; text-transform: none; 
	 white-space: normal; widows: 2; word-spacing: 0px; -webkit-text-stroke-width: 0px; 
	 background-color: #ffffff; text-decoration: underline; ">交通网络分析服务提供者配置参数</a> ；选择发布为三维网络分析服务时，参数说明请参考：<a class="contentpagehyperlink" href="../../../Server_Service_Management/GISService_Config/Service_Config/Config_Provider/ProviderParameter/UGCFacilityAnalyst3DProvider.htm">三维网络分析服务提供者配置参数</a>；</li>
    <li class="ListInParagraph">等待上传结束、服务发布成功，即可完成此流程；若在上传数据时未选择“上传并发布”而选择直接“上传”，您也可以等待上传完成后，在“资源中心”&gt;“数据”&gt;“我的数据”页面执行发布操作。</li>
  </ol>
  <p class="myNormal" style="color: #000000;">注意：若上传 ShapeFile 类型的数据，还需选择文件编码格式： UTF-8/GBK。</p>
  <p class="myNormal" style="color: #000000;"> </p>
  <p class="myNormal" style="color: #000000;">上传成功后，系统会自动跳转到该项数据资源的详情页面，您可查看或编辑相关信息、已发布的服务（若勾选了发布），也可下载数据。若有共享的需求，您可切换到该项数据资源的“共享设置”面板修改共享信息，详见：<a class="contentpagehyperlink" href="../share_resource/share_dataResource.htm">共享数据</a>。</p>
  <p class="myNormal" style="color: #000000;">上传后的数据均显示在“资源中心”&gt;“数据”&gt;“我的数据”页面。除了发布、编辑、下载、共享等功能，在“我的数据”页面，您还可进行更新数据、删除数据、预览pdf数据的操作。</p>
  <ul class="SymbolList1" style="list-style: disc;">
    <li class="SymbolList1">若待更新数据的数据已发布了服务，更新数据后服务会重新发布，但服务名称会保持不变。</li>
    <li class="SymbolList1">若删除数据后，发布的服务也会同步删除。<span style="font-weight: bold;">注意：</span>若数据有关联的地图、洞察、大屏、应用等，删除数据会导致这些关联资源不能正常访问。更多关于删除，详见：<span><a class="contentpagehyperlink" href="../delete_resource/delete_resource.htm">删除资源</a></span>。</li>
  </ul>
