<p class="myNormal">作为门户软件平台，除核心的 GIS 功能模块外，iPortal 也提供了一些通用的功能模块，如新闻中心、意见反馈，以满足用户实际项目的需求。</p>
<h2><a id="新闻中心"></a>新闻中心</h2>
<p class="myNormal">iPortal 为管理员提供了<a class="contentpagehyperlink" href="../iportal_management/news.htm">新闻管理模块</a>，管理员可添加、编辑、发布新闻。新闻发布后，用户可以通过首页导航栏进入“新闻中心”，查看门户中已发布的新闻通知。</p>
<h2>意见反馈</h2>
<p class="myNormal">在使用门户产品过程中，用户除了可以通过常规方式，如电话、邮件、在线 QQ 等方式将问题或意见反馈给门户管理员，也可通过 
 iPortal 提供的“意见反馈”功能进行在线反馈。</p>
<p class="myNormal">在 iPortal 首页底部点击“意见反馈”进入意见反馈页面，普通用户（包括匿名用户）可浏览、搜索所有的意见反馈项，也可点击右侧的“我要反馈”，填写以下信息，反馈新的功能问题或改进意见。</p>
<ul style="list-style: disc;" class="SymbolList1">
	<li class="SymbolList1">姓名：您的姓名。匿名用户必须填写；对于登录用户，系统会自动使用您的昵称进行填充。</li>
	<li class="SymbolList1">电子邮箱：选填。为了便于产品团队或技术支持与您联系，建议填写。</li>
	<li class="SymbolList1">标题：您要反馈的问题或意见标题，必填项。</li>
	<li class="SymbolList1">内容：您要反馈的问题或意见的详细描述，必填项。</li>
	<li class="SymbolList1">验证码：不区分大小写，必填项。</li>
</ul>
<p class="myNormal">填写完成后，点击“提交反馈”即可完成反馈。新提交的反馈项经管理员审核后，才会在反馈列表中进行展示，且反馈项仅支持管理员进行回复，详见：<a class="contentpagehyperlink" href="../iportal_management/feedback.htm">意见反馈管理</a>。</p>
