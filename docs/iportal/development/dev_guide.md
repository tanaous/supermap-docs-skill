  <p class="myNormal">SuperMap iPortal 为广大用户提供了整套 SDK，应用开发人员可以使用 SuperMap iPortal 提供的 Web 服务接口开发需要的应用系统。在 <b><a class="contentpagehyperlink" href="Use_iPortal_RestAPI/Introduction.htm">使用 iPortal REST API</a></b> 部分提供了丰富的示例介绍 SuperMap iPortal REST API 的使用，利用 REST API 实现了一些操作，包括对地图的操作、对数据的操作、对群组的操作等。</p>
  <p class="myNormal">SuperMap iClient JavaScript 提供了对接 SuperMap iPortal 中地图、数据、服务等资源的 API，帮助您在自己的 Web 应用中获取或更新 iPortal 中的资源。</p>
  <p class="myNormal">同时，SuperMap iPortal 还提供了一套深度定制开发指南，协助应用开发人员根据自身业务需求，量身定制一系列符合用户实际应用的门户功能，快速构建 GIS 门户软件平台。</p>
  <p class="myNormal">关于  SuperMap iPortal 所提供的 SampleCode 的总体浏览和介绍，请参考<a class="contentpagehyperlink" href="ip_SampCode_Intro.htm">示范程序说明</a>。</p>
